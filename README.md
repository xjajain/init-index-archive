# init-index-archive

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面可以用于展示、测试、存档或作为静态资源对外提供访问。仓库本身不绑定具体域名或网站，仅作为通用 HTML 页面集合的存储与分发点。

## 目录说明

- `index/` — 存放主要 HTML 页面文件（可包含子目录）
- `assets/` — 存放页面引用的 CSS、JS、图片等静态资源
- `archive/` — 存放旧版本或不再维护的页面备份
- `README.md` — 本文件，仓库说明

## 页面归档说明

所有归档页面均以独立文件形式存放，文件名应体现页面内容或用途。每个页面建议附带简短说明（可写在页面注释或 `index/` 目录下的 `README.md` 中）。  
新增页面时，请遵循以下原则：

- 页面本身应完整自洽，不依赖外部不可控资源
- 尽量使用相对路径引用资源文件
- 避免包含恶意、侵权或违反 GitHub 服务条款的内容

## 维护说明

- 本仓库由维护者不定期更新，不保证更新频率
- 欢迎通过 Issues 提交建议、报告问题或提出页面归档请求
- 如需贡献页面，请 Fork 本仓库后提交 Pull Request
- 页面内容由贡献者自行负责，维护者不承担内容责任

## 使用方式

你可以直接通过 GitHub Pages 或其他静态托管服务访问本仓库中的 HTML 页面。具体访问路径取决于部署配置。  
若需本地预览，请克隆仓库后直接在浏览器中打开对应 HTML 文件。
